# 📄 AI-Based Resume Screening System
📌 Project Overview

This project is a web-based Resume Screening System developed using Machine Learning and Natural Language Processing (NLP). The system helps automate the process of analyzing and screening resumes based on job-related skills and requirements.

The main goal of the project is to reduce manual effort in recruitment by identifying suitable candidates more efficiently.

## 🎯 Objective
Automate resume screening process
Extract and analyze resume content
Match resumes with required job skills
Rank or filter candidates based on relevance

## ⚙️ Features
Upload resumes through web interface
Process resume text using NLP techniques
Match resumes with job-related keywords
Display candidate screening results
Simple and user-friendly interface

## 🔍 Workflow
### 1️⃣ Resume Upload

Users upload resumes through the web application.

### 2️⃣ Text Processing

The system extracts important text and skills from resumes.

### 3️⃣ Skill Matching

Machine Learning techniques are used to compare resumes with required job skills.

### 4️⃣ Candidate Screening

Candidates are ranked or filtered based on skill matching.
